Research Projects

Indoor Pedestrian Dead Reckoning (PDR) System

2024 – Present

Problem: Indoor positioning without GPS drifts over time, and results change depending on how the user holds the phone.

Approach: Built step detection for 3 phone modes (hand/pocket/swing) using a state machine and quaternion transforms in C++17/JNI. Tested 5 stride formulas and 7 detection methods on 700+ datasets.

Result: Pattern-based step detection reached 97% accuracy with 1% false positive rate. Compared 5 stride formulas on 700+ datasets from 8 participants across 3 walking speeds.

Kotlin C++17 Python NDK/JNI FFT Quaternion

Multi-Building Indoor Navigation Platform (Team project)

2024 – Present

Problem: No single system could handle navigation across multiple buildings and floors with real-time floor detection.

Approach: Built ZeroMap apps (compass + PDR), floor detection using air pressure sensors, and a C++ simulator (38K LOC) with OpenCV for visual comparison.

Result: Tested in 4 buildings (KU campus, Daewoo apartment, Suwon station, naval vessel) with 3 simulators and Express.js / Spring Boot servers.

Android C++17 OpenCV Express.js Spring Boot Unity 3D

Real-time Indoor Object Detection & OCR

2025 – Present

Problem: PDR-only navigation requires a known starting position, but there is no way to determine the initial location indoors without manual input.

Approach: Trained YOLOv8n (8 classes) to detect indoor signs and objects on Android (ONNX Runtime + Camera2), then used OCR to read sign text and match it against POI data already stored in the indoor map to estimate the user's initial position.

Result: 8-class detection model achieved mAP50 80.8% (Precision 87.3%), running in real time on mobile.

YOLOv8 ONNX Runtime Kotlin Camera2 API

Automated Floor Plan Processing & Indoor Map Generation (Team project)

2024 – Present

Problem: Turning CAD floor plans (DXF) into indoor map data by hand is slow and does not scale.

Approach: Built a Python tool that reads DXF files, finds stairs by shape, and uses YOLOv8 to detect elevators and exits from floor plan images.

Result: Automatically extracted maps for 4 buildings (Hansando, Daewoo, Suwon Station, Anam Hospital) without manual work.

Python ezdxf YOLOv8 Shapely
